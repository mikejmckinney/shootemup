## Purpose
Level-up choice screen showing 3 upgrade options with clear icons and build agency.

## Composition Notes
Three floating panels centered. Each shows weapon/passive upgrade with icon, minimal label. Dimmed gameplay in background. Clear primary action.

## Prompt
Level-up upgrade selection UI mockup for modern arcade shooter. HD 2D vector-style illustration, clean geometric shapes, flat color surfaces with subtle gradients, smooth edges. Three floating upgrade panels arranged horizontally, center screen. Each panel: clean geometric icon (weapon or passive symbol), minimal label text placeholder, subtle glow on hover state. Dimmed gameplay background visible. Player ship silhouette faintly visible below. One primary action button. Icon-first communication. Geometric sans typography feel. Consistent corner radius and stroke width. Generous spacing. Dark background with bright UI elements. Premium confident presentation. Minimalist sci-fi aesthetic.

## Negative Prompt
pixel art, retro style, texture noise, grit, cluttered UI, text-heavy, decorative frames, small buttons, realistic, painterly, busy background

## Consistency Constraints
- UI panel style: floating panels, consistent corner radius, stroke width
- Icon style: clean geometric symbols
- Color palette: dark background, bright UI elements, cyan accent
- Typography style: geometric sans feel, all-caps headers, minimal text
