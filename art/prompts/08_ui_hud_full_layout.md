## Purpose
Complete HUD layout reference showing all component positions during gameplay.

## Composition Notes
Full screen HUD overlay on gameplay. HP/shield top-left. Score/timer/objective top-right. Weapons bottom-left. Abilities bottom-right. Optional boss HP top-center.

## Prompt
Full HUD layout mockup for modern arcade shooter. HD 2D vector-style illustration, clean geometric shapes, flat color surfaces with subtle gradients, smooth edges. Complete UI overlay on dimmed gameplay background. Top-left: HP bar with shield indicator, status icons. Top-right: score counter, timer, objective icon. Bottom-left: weapon slots with cooldown indicators. Bottom-right: dash ability icon, active ability, consumable slots. Top-center: boss HP bar with phase markers (optional element shown). All elements as floating panels. Icon-first design. Large legible numbers. Minimalist sci-fi HUD style. Consistent corner radius. Grid-aligned layout. Dark background, bright UI. Readable at a glance.

## Negative Prompt
pixel art, retro style, texture noise, cluttered UI, text-heavy, decorative borders, small elements, realistic, painterly, inconsistent styling

## Consistency Constraints
- HUD layout: HP top-left, score/timer top-right, weapons bottom-left, abilities bottom-right, boss HP top-center
- UI panel style: floating panels, consistent corner radius
- Icon style: clean geometric symbols
- Color palette: dark background, bright UI elements
- Typography style: geometric sans, large legible numbers
