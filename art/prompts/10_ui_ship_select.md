# 10_ui_ship_select

## Metadata
- **Filename**: 10_ui_ship_select.png
- **Category**: UI/UX Mockups
- **Size**: 1536x1024

## Purpose
Ship selection screen showing 3-4 ships with trait previews and visual differences.

## Composition Notes
Ships displayed as selectable cards. Each shows silhouette, trait icon, minimal stats. Selected ship enlarged or highlighted. Start button prominent.

## Prompt
Ship selection UI mockup for modern arcade shooter. HD 2D vector-style illustration, clean geometric shapes, flat color surfaces with subtle gradients, smooth edges. Four ship selection cards arranged horizontally. Each card: ship silhouette with distinct shape (angular, bulky, sleek, asymmetric variants), trait icon below, minimal stat indicators. Selected ship card enlarged with cyan highlight border. Large Start button as primary action. Dark navy background. Floating panel cards. Icon-first communication. Ships show strong silhouette differences. Premium sci-fi aesthetic. Generous spacing. Readable at a glance.

## Negative Prompt
pixel art, retro style, texture noise, identical ships, cluttered UI, text-heavy descriptions, decorative frames, small elements, realistic, painterly

## Consistency Constraints
- Ship silhouette variations: angular, bulky, sleek, asymmetric distinct shapes
- UI panel style: floating panel cards, consistent corner radius
- Icon style: trait icons below ships
- Color palette: dark navy background, cyan highlight for selected
- Typography style: minimal stat indicators
