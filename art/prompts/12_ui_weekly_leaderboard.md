## Purpose
Weekly challenge leaderboard showing competitive ranking with fair-play indicators.

## Composition Notes
Leaderboard list with rank, name placeholder, score. Current player position highlighted. Top 3 with distinct styling. Challenge info at top.

## Prompt
Weekly challenge leaderboard UI mockup for modern arcade shooter. HD 2D vector-style illustration, clean geometric shapes, flat color surfaces with subtle gradients, smooth edges. Large floating panel. Top section: Weekly Challenge title with timer countdown, standardized loadout icon indicating fair play. Leaderboard list: rank numbers, player name placeholders, scores. Top 3 positions with gold/silver/bronze accent colors. Current player row highlighted with cyan border. Global/Friends tab toggle. Score displays large legible numbers. Minimalist sci-fi panel style. Dark background. Fair competition visual emphasis. Premium confident presentation.

## Negative Prompt
pixel art, retro style, texture noise, cluttered list, tiny text, decorative frames, realistic avatars, painterly, busy background

## Consistency Constraints
- UI panel style: large floating panel
- Icon style: standardized loadout icon, tab toggle
- Color palette: dark background, gold/silver/bronze accents, cyan highlight
- Typography style: large legible numbers
- Leaderboard style: rank, name placeholder, score format
