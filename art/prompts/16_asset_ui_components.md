## Purpose
UI component kit showing panels, buttons, icons, and HUD elements for consistency.

## Composition Notes
Component library format. Panels, buttons (states), icons, bars, indicators. Consistent corner radius and stroke weight throughout.

## Prompt
UI component asset sheet for modern arcade shooter. HD 2D vector-style illustration, clean geometric shapes, flat color surfaces with subtle gradients, smooth edges. Dark navy background. UI kit layout: floating panels (various sizes), buttons (normal/hover/pressed states), icon set (weapon types, abilities, status effects, rewards), progress bars (HP, shield, cooldown), indicators (score counter frame, timer frame). Consistent corner radius. Consistent stroke weight. Minimalist sci-fi style. Cyan accent for player elements. Gold for rewards. Red for warnings. Component library format. Premium clean presentation.

## Negative Prompt
pixel art, retro style, texture noise, inconsistent styling, decorative frames, cluttered, realistic, painterly, text-heavy

## Consistency Constraints
- UI panel style: floating panels, consistent corner radius, stroke weight
- Button style: normal/hover/pressed states
- Icon style: weapon types, abilities, status effects, rewards
- Color palette: cyan for player, gold for rewards, red for warnings
