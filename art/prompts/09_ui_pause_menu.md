## Purpose
Pause menu showing navigation structure with clear button hierarchy.

## Composition Notes
Centered floating panel. Resume as primary action. Options, exit as secondary. Blurred/dimmed gameplay behind. Clean, minimal options.

## Prompt
Pause menu UI mockup for modern arcade shooter. HD 2D vector-style illustration, clean geometric shapes, flat color surfaces with subtle gradients, smooth edges. Central floating panel with rounded corners. Resume button as large primary action with highlight state. Secondary buttons: Options, Exit stacked below. Blurred dimmed gameplay visible in background. Minimalist sci-fi panel style. Icon paired with each option. Geometric sans typography placeholder. Large touch-friendly buttons. One primary action prominent. Clear hover/press state indication. Dark background, bright UI elements. Premium confident presentation.

## Negative Prompt
pixel art, retro style, texture noise, cluttered menu, many nested options, decorative frames, small buttons, realistic, painterly, text-heavy

## Consistency Constraints
- UI panel style: floating panels, consistent corner radius
- Button style: large touch-friendly, clear states
- Icon style: paired with options
- Color palette: dark background, bright UI elements
- Typography style: geometric sans placeholder
